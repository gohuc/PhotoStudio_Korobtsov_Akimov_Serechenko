# TaskTracker

Автор: Коробцов Иван, Сереченко Анастасия, Акимов Илья, группа 25 ИС-11Д

## Описание
Учебный проект: Фото студия.

## План функций
- Добавить задачу
- Показать список задач
- Изменить статус задачи
- Удалить задачу

## Стек
C# / .NET / Visual Studio

## Как запустить
1) Открыть solution в Visual Studio
2) Нажать F5 (Run)
3) В консоли появится меню
