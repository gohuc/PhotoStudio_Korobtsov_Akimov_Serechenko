# Отчёт ЛР2 — TaskTracker

## Что сделано
- Добавлен проект PhotoStudio.Core
- Создана модель TaskItem и enum TaskStatus
- Реализован TaskService (Add/GetAll)
- В PhotoStudio.App добавлены команды:
  - Добавить задачу
  - Показать список

## Как проверить
1) Запустить программу (F5)
2) Нажать 2 → список пуст
3) Нажать 1 и ввести пусто → ошибка
4) Нажать 1 и ввести "Купить тетрадь" → задача добавлена
5) Нажать 2 → задача отображается
